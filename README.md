## 计算机网络性能
### 速率
    数据率，比特率，数据传输速率
    b/s(bps),kb/s,mb/s,gb/s
    往往指额定速率，标称速率
### 带宽
    通信领域：最高和最低频率之差（hz）
    网络：数字信道所能传输的“最高数据率”bps
### 延迟/时延
    结点处理延迟 d1
    排队延迟 d2 流量强度=La/R
    传输延迟（分组长度bits）L，链路带宽bps R d3
    传播延迟：物理链路长度，信号传播速度 d4 ！！
### 时延带宽积
    时延带宽积=传播时延*带宽（以比特为单位的链路长度）

### 分组丢失（丢包）
    丢包率=丢包数/已发射分组数
### 吞吐量/率
    b/s
    即使吞吐量
    平均吞吐量
    瓶颈链路（取决于小的吞吐率）
